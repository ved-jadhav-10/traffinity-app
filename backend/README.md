# Backend Configuration

This folder contains backend configuration files for the Traffinity app.

## Environment Variables

The `.env` file contains sensitive configuration data:

- Supabase Project URL
- Supabase Anon Key

**Important:** The `.env` file should be added to `.gitignore` to prevent accidentally committing sensitive data.

## Files

- `.env` - Actual environment variables (git-ignored)
- `.env.example` - Example template for environment variables
- `README.md` - This file
