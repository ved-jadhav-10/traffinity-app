# Font Installation Instructions

To complete the splash screen setup, you need to download the Cormorant SC font files:

## Steps:

1. Visit [Google Fonts - Cormorant SC](https://fonts.google.com/specimen/Cormorant+SC)
2. Click "Download family" button
3. Extract the downloaded ZIP file
4. Copy the following files to this folder (`assets/fonts/`):
   - `CormorantSC-Light.ttf`
   - `CormorantSC-Regular.ttf`
   - `CormorantSC-Medium.ttf`
   - `CormorantSC-Bold.ttf`

5. Download **Poppins** font from [Google Fonts - Poppins](https://fonts.google.com/specimen/Poppins)
6. Copy these Poppins files to this folder:
   - `Poppins-Regular.ttf`
   - `Poppins-Bold.ttf`

5. After adding the fonts, run:
   ```
   flutter pub get
   ```

6. Then run your app:
   ```
   flutter run
   ```
